Lists in HTML

Unordered List The list starts with the <ul> tag. Each list item starts with the <li>

Ordered List An ordered list starts with the <ol>tag. Each list item starts with <li> the tag.

Description List A description list is a list of terms, with a descripton of each term. The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term.

Lists in CSS

With CSS it is possible to change the List Item Markers to a different style.

Tables in HTML

An HTML table is defined with the <table> tag. Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the tag <td> tag.

Tables in CSS

With CSS it is possible to give a table a border. It is also possible to define a width and height for the table with CSS. With the padding property it is possible to control the space between the border and the content in a table. With the :hover selector on tr it is possible to highlight table rows on mouse over.
